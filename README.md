# Photo gallery

A simple gallery for the job interview.


### Prerequisites
docker and docker-compose

## How to run?
 - docker-compose up
 
 or
 
 - ./gradlew bootRun (only if you have a locally running database)
 
## How to use?
go to: http://localhost/swagger-ui.html
 
## Used technologies
 - Java 8
 - Spring Boot 2
 - PostgreSQL
 - Lombok
 - Swagger
